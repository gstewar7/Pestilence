If you want try demo level, just download and import to project "Standart Assets" from Unity Asset Store - https://assetstore.unity.com/packages/essentials/asset-packs/standard-assets-32351

Drag FPS Controller prefab to scene and set tag "Player".

Doors and windows use simple script working when player cross a trigger and press key "E".
